# companies_context_v1.md — 3社事業文脈（第2面プロンプト用）

## このドキュメントについて

第2面「社長の朝会」のコメント自動生成で、LLM プロンプトに動的注入する社別の事業文脈。3社それぞれについて、ウォッチすべき軸、優先論点、避けたい煽りパターン、競合リストを構造化する。

LLM はこの文脈を読んだ上で、各社のニュース記事に対して「今朝の問い」（40〜80字）を生成する。

参照元：

- `news_profile.md` §4.2.1〜§4.2.3（神山さんの3社別ウォッチ優先順位・選定方針の原典）
- `sources/companies.md`（3社別ソース定義、High/Medium/Reference 区分）
- `docs/blacklist_v1.md`（避けたい煽りパターンのキーワード辞書、美意識4 連動）

---

## 1. Cocolomi（生成AI導入支援）

### 事業の本質

日本企業の生成AI導入を支援する事業。自社規模では大手競合の動向を参照軸にしない方針で、競合追随ではなく **「市場の地殻変動」を捉える** ことに価値の重心を置く。お金が動く場所（投資・予算）、ルールが決まる場所（規制）、現場で起きていること（導入事例）——この3つの動きを読み解いて顧客に届ける情報の質で差別化する立場（news_profile.md §4.2.1）。

### ウォッチすべき軸（優先順位順）

1. AI事業者ガイドライン、AI戦略会議、規制動向
2. 大手企業の生成AI導入事例（成功・失敗両方）
3. AI技術の進化（モデル性能、ツール、フレームワーク）
4. 生成AIと著作権・個人情報の論点
5. 企業の生成AI予算・投資動向（DX推進企業の意思決定タイムライン）

優先順位は news_profile.md §4.2.1 の「高」「中」「低」分類に概ね対応する。1〜2 は「高」（規制・事例）、3 と 5 は「高」「中」の境界（技術進化・予算動向）、4 は規制隣接論点。

### 「今朝の問い」の方向性

- 自社事例研究との接続を促す問い
- ガイドライン適応の好機を見逃さない問い
- 顧客対話で使える問い

### 避けたい記事の傾向

- AI画像生成、ゲームAI、TikTok AI 等の B2C 文脈
- AI 業界の煽り記事（「AGI 到来」等）
- 単なる新サービス発表（事業含意がないもの）

`docs/blacklist_v1.md` の「成功談・カリスマ礼賛系」「誇張表現系」「金額誇示系」と概ね重なる。例：「驚異の○○」「業界震撼」「○億円調達」がタイトルに入る記事は LLM 評価で低スコアにする。

### 競合・類似事業者（記事に登場したら注意）

- ABEJA
- ELYZA
- エクサウィザーズ（Exa Wizards）
- PKSHA（パークシャ）
- AI inside
- Cinnamon AI
- Preferred Networks
- Lightblue

これらの記事は「同業の動き」として扱う。news_profile.md §4.2.1 の方針「競合・類似事業者の動向は弾く——自社規模では参照軸にならない」に従い、Stage 1 では `scripts/selector/hard_filter.py` の `_COCOLOMI_COMPETITORS` で機械的にハードフィルタ済。第2面プロンプトに記事が届いた場合（フィルタ漏れ）は、コメントで「参照軸ではない」旨を明示する。

## 2. Human Energy（企業向け研修）

### 事業の本質

企業向け研修サービス事業。**HRテックや大企業人事の流行りとは戦わず、現場の本質課題と理論的深みの両端で勝負する** 戦略観を持つ。研修手法・教育理論・組織開発論考・心理学／認知科学が差別化の源泉。同業情報は「手法・教育理論として学べるもの」だけを拾い、営業戦略・市場シェア争い・拡大路線の情報は弾く（news_profile.md §4.2.2）。

### ウォッチすべき軸（優先順位順）

1. パーパス経営、心理的安全性、エンゲージメント
2. 越境学習、リスキリング、対話型組織開発
3. 心理学・認知科学の応用（Edgar Schein、ディープリスニング）
4. 海外L&D動向（MIT Sloan、HBR）

news_profile.md §4.2.2 の「高」（業界全体動向 / クライアント業界の経営課題 / 組織開発・カルチャー変革論考）と「中」（研修手法・教育理論 / 心理学・認知科学研究 / 海外研修・人材開発トレンド）の優先項目を、神山さんの実務語彙に翻訳した形。

### 「今朝の問い」の方向性

- 研修プログラム改善のヒント
- 顧客企業の課題への新しい切り口
- 研修業界の地殻変動

### 避けたい記事の傾向

- 根性論、感動研修、単発研修
- 「○○式マネジメント」のカリスマ礼賛
- 研修効果の単純な数値化

`docs/blacklist_v1.md` の「個人崇拝系」「ハック系」「断定強調系」と重なる。例：「○○氏の経営哲学」「最強の○○術」「これだけやれば」がタイトル中心の記事は弾く。さらに news_profile.md §4.2.2 の選定方針通り、**HRテック流行り・大企業人事の流行り・同業の営業戦略／拡大路線** はテーマレベルで除外（hard filter は具体キーワード難で空、Stage 2 LLM 評価に委ねる）。

### 競合・類似事業者（記事に登場したら注意）

現在、`scripts/selector/hard_filter.py` の `_HUMAN_ENERGY_COMPETITORS` は **空リスト**。同業の営業戦略・拡大路線は具体キーワードで弾きにくいため、Stage 2 LLM 評価でテーマレベルの判断に委ねる方針（同モジュール内コメント参照）。

## 3. Web-Repo（フランチャイズ業界）

### 事業の本質

フランチャイズ業界の専門メディア事業（運営25年、日本最大級）。**グループ3社の中で唯一、業界内に大きなプレゼンスを持つメディア事業**。目的は β（自社ビジネスとしての経営判断）——ニュース取り上げ判断、広告主獲得、事業開発の判断材料を得る。news_profile.md §4.2.3 の優先順位は「加盟者・オーナー視点」を最上位に据え、業界別FC動向と規制・法改正・判例を中心に追跡する。

### ウォッチすべき軸（優先順位順）

1. FC契約改正、加盟者保護、独占禁止法、JFA動向
2. 24時間営業見直し、FC本部経営難
3. 飲食・小売・サービス業の FC 動向
4. 新興 FC モデル（社会課題解決型、サブスク型）

news_profile.md §4.2.3 の「高」（FC加盟者・オーナーの動向 / 業界別FC動向 / 規制・法改正・判例）と「中」（FC本部企業の動向 / 新興FCモデル）を実務軸に展開した形。

### 「今朝の問い」の方向性

- ウェブリポメディアでの取り上げ判断
- 業界の地殻変動を読む観点
- 加盟者保護の論点

### 避けたい記事の傾向

- 「楽して儲かる」「副業で月○万円」「未経験OK」
- 脱サラ・自由な働き方の煽り記事
- 成功者インタビュー一辺倒

`docs/blacklist_v1.md` の「金額誇示系」「成功談系」「断定強調系」と重なる。例：「年商○億円」「○ヶ月で○倍」「成功者」「成功法則」がタイトルに入る記事は弾く。news_profile.md §4.2.3 の「低」優先（同業メディア / 海外FC / 関連経済トレンド）も第2面採用は基本見送り。

### 競合・類似事業者（記事に登場したら注意）

- アントレ
- フランチャイズの窓口
- フランチャイズ比較ネット

## 4. プロンプト注入時の使い方

LLM プロンプトには、対象記事が属する **1社分のセクションのみ** を動的に注入する。3社全ては注入しない（プロンプト肥大化を防ぐ）。

例：Cocolomi 関連記事のコメント生成時 → §1 Cocolomi のみを注入。

### 記事と社の対応付け

`scripts.lib.source.Source.category` の値で判定する。`sources/companies.md` のソースは以下の category を持つ：

- `companies:Cocolomi` → §1 を注入
- `companies:Human Energy` → §2 を注入
- `companies:Web-Repo` → §3 を注入

業界横断ニュース（例：generic な経済記事）が第2面の対象になる場合の運用は Sprint 2 で別途検討する（暫定：3社いずれかの category に紐付かない記事は第2面の対象外とし、第3面以降に振る）。

### 注入時の整形ルール

- セクションの末尾「---」区切りまでをそのまま挿入する
- 章番号「## 1. Cocolomi」等は注入時に「## 事業文脈」へ書き換えて統一する（プロンプト内では1社のみなので章番号不要）
- 参照元注記（"news_profile.md §4.2.1" 等）は残す——LLM が出典の重みを認識しやすい
