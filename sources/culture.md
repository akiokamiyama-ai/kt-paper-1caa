# Culture — 日本語カルチャー領域の情報源

このファイルは Kamiyama Tribune **第5面 セレンディピティ枠（Page V）** の情報源リストである。
神山さんの「**普段の関心領域から離れた、予断なく見る**」枠として、Z世代カルチャー、デザイン、メンズファッション、アート・カルチャーを横断的に拾う。

選定の方針：**知的読者が読める日本語カルチャーメディア**を選ぶ。Z世代向け軽量バイラルメディア（TikTok まとめ等）、美容・コスメ専門誌、シニア向け女性誌、アイドル/芸能ゴシップ専門は含めない。**Medium 2 ソース**（TABI LABO、AXIS）で主力供給、**Reference 2 ソース**（houyhnhnm、cinra.net）で多様性確保。Reference は記事数が多く玉石混淆のため Stage 2 評価による品質フィルタの二段構えで対応する。

選定の上位基準は `news_profile.md` §3（美意識7項目）と §5（編集ポリシー）。特に「**目利き感覚で主流外の本質を発掘する**」（美意識2）と「**構造と細部の往復**」（美意識1）を意識すること。

**ステータス凡例**：
- ✅ 検証済み（フィード/URLが実際に取得できることを確認）
- ⚠️ 一部未検証
- ❌ 未検証

## カルチャー主力（Medium）

### 1. TABI LABO ✅
- **URL**: https://tabi-labo.com/
- **RSS**: https://tabi-labo.com/feed/
- **形式**: RSS 2.0
- **mainstream**: true
- **対象**: Z世代カルチャー、社会観察、ライフスタイル観察、TikTok 発トレンド分析、若者の消費行動・価値観の変化
- **位置付け**: **神山さんの普段の関心領域から離れた「予断なく見る」枠の主力**。経営者として若年層の価値観・消費行動を把握する経路としても機能する。**更新頻度は1日3〜4本（2026-05-03 計測：直近5本が48時間以内）**

### 2. AXIS ✅
- **URL**: https://www.axismag.jp/
- **RSS**: https://www.axismag.jp/feed
- **形式**: RSS 2.0
- **mainstream**: false
- **対象**: プロダクトデザイン、建築、グラフィックデザイン、家具・照明・カメラ等の製品哲学、デザイナーインタビュー
- **位置付け**: **神山さんの美意識1（構造と細部の往復）と直結する硬派なデザイン媒体**。日本のデザイン誌『AXIS』Web 版で、製品紹介に留まらず作り手の思想を掘る記事が多い。news_profile.md §3 美意識1（細部に作り手の哲学が宿る）の英語圏外での重要ソース。**更新頻度は1日2〜5本（2026-05-03 計測：5/1 に5本投下）**

### 3. WIRED.jp ✅
- **URL**: https://wired.jp/
- **RSS**: https://wired.jp/rssfeeder/
- **形式**: RSS 2.0
- **mainstream**: true
- **対象**: テクノロジー × カルチャー、Condé Nast Japan 運営。Business / Culture / Mobility / Science / Health / Opinion を横断的にカバー。AI 規制、テック企業動向、サイエンス、社会論考も射程
- **位置付け**: WIRED.com の日本版だが、独自編集・翻訳記事が多く、**日本語で読めるテック × カルチャー長文ジャーナリズム**として culture 領域で機能。神山さんの「予断なく見る」枠で、デザイン (AXIS) / 若者観察 (TABI LABO) と並ぶ第3の軸。**更新頻度は1日3〜5本（2026-05-03 計測：日次4本ペース）**。default UA で 200 を返すため site_overrides 不要

## カルチャー多様性確保（Reference）

### 4. houyhnhnm ✅
- **URL**: https://www.houyhnhnm.jp/
- **RSS**: https://www.houyhnhnm.jp/feed
- **形式**: RSS 2.0
- **mainstream**: false
- **対象**: メンズファッション + カルチャー、20年以上の老舗（フイナム）。新作リリース、ブランド別注、編集者連載、ショップオープン情報、ファッション横断のカルチャー記事
- **位置付け**: **記事数多いため Reference 配置で priority 抑制**。ファッション業界の通常ニュース（新作・別注・店舗情報）が大半だが、編集者連載や特集記事には批評性のあるものも混ざる。**Stage 2 評価で品質の高い記事を選別する設計**。**更新頻度は1日5本以上（2026-05-03 計測：5/2 だけで5本超）**

### 5. cinra.net ✅
- **URL**: https://www.cinra.net/
- **RSS**: https://www.cinra.net/feed
- **形式**: RSS 2.0
- **mainstream**: true
- **対象**: アート・カルチャー全般。映画・音楽・展覧会・舞台のニュース、批評記事、アーティストインタビュー
- **位置付け**: **批評性高い記事もあるが、芸能ニュース寄り（ドラマキャスト発表、リリース告知）も混在する**。Reference 配置 + Stage 2 評価による品質フィルタの二段で対応。批評記事・インタビューが入った日は質が高い、という非対称な扱い。**更新頻度は1日5本以上（2026-05-03 計測：5/1 だけで3本、芸能告知含む）**
